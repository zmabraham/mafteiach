# Mafteiach - The Rebbe's Torah Index

A complete mirror and database of the Lubavitcher Rebbe's Torah teachings from [mafteiach.app](https://www.mafteiach.app).

## 🌐 Live Site

**[https://zmabraham.github.io/mafteiach/](https://zmabraham.github.io/mafteiach/)**

## 📊 What's Included

### 1. Complete Website Mirror
- **3,404 HTML pages** from mafteiach.app
- **3,331 Farbrengen entries** with full content
- All navigation and browsing functionality
- Original site structure preserved

### 2. Database Search App
- **3,350 entries** in a searchable database
- Filter by: content type, Hebrew month, year
- Full-text search in Hebrew and English
- Direct links to mafteiach.app originals

## 🚀 Features

### Mirror Site (`/`)
- Browse by year, month, parsha, occasion
- All farbrengen entries
- Likutei Sichos, Maamarim sections
- Original mafteiach.app experience

### Database App (`/database-app/`)
- Fast full-text search
- Advanced filtering
- Responsive design
- Works offline (embedded data)

## 📦 Project Structure

```
├── index.html              # Mirror home page
├── all/                    # Browse by year/month/parsha
├── farbrengen/             # 3,331 entry pages
├── likkutei_sichos/        # LS sections
├── maamorim/               # Maamarim
├── database-app/           # Searchable database app
├── mafteiach.db            # SQLite database
└── generate_frontend.py    # Database app generator
```
